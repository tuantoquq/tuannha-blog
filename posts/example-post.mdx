## Show the output of code

To show the output of in code, you can use the following syntax:

1. Python

```python

def hello():
  print("Hello World")

```

2. JavaScript

```javascript
const hello = () => {
  console.log('Hello World')
}
hello()
```

3. Java

```java
public class HelloWorld {
  public static void main(String[] args) {
    System.out.println("Hello, World");
  }
}
```

4. Go

```go
package main

import "fmt"

func main() {
    fmt.Println("hello world")
}
```
